# Jupiter Wallet

React based Jupiter frontend.

## Usage

**Install deps**

```sh
yarn install

yarn husky_setup
```

**Start**

```sh
yarn start
```

**Build**

```sh
yarn build
```

**Test**

```sh
yarn test
```
